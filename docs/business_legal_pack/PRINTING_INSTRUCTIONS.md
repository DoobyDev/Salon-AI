# Printing Instructions

## Quick Print

1. Open a file in VS Code preview or browser.
2. Print to PDF or paper (`Ctrl+P`).
3. Use A4 paper, portrait orientation.
4. Enable "Background graphics" for HTML print packs.

## Recommended Print Order

1. `print/LEGAL_PRINT_PACK.html`
2. `print/BUSINESS_PLAN_PRINT_PACK.html`
3. Individual markdown docs if you need separate signature packs.

## Signature Pages

- Add signature blocks where required after legal review.
- Keep one signed master copy and one operational copy.
